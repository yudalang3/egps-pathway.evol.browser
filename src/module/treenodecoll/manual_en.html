Tree Node Info Collector

Collect leaf/internal node information from a phylogenetic tree (NWK and related formats).

1. Quick Start

  1. Click Import to load a phylogenetic tree.
  2. Configure the parameters (use Example to check supported formats).
  3. Click Execute to get the collected information.

2. Output

  • Leaf node list (IDs/names and optional metadata).
  • Internal nodes (when enabled) and their basic properties.

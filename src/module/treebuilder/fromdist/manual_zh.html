进化树构建器：从进化距离矩阵

使用距离法从预先计算的进化距离矩阵重建系统发育树。

1. 快速开始

  1. 在参数输入区域查看所需输入数据
  2. 输入所需参数
  3. 点击 执行 查看结果

2. 功能概述

本模块用于根据已计算好的进化距离矩阵重建系统发育树。支持三种距离法构树算法（Neighbor-Joining、Swift Neighbor-Joining、UPGMA），并在构树后自动用 mid-point 方法定根。结果会在 Modern Tree Viewer 中打开。

3. 输入格式

输入文件为 eGPS 距离矩阵文本格式（下三角含对角线）。空行或以 # 开头的注释行会被忽略。

3.1 格式示例

# 物种/序列个数
3
sp1 0
sp2 0.12 0
sp3 0.20 0.18 0

3.2 格式规则

  • 第一行是物种/序列数量 N
  • 之后每行是名称 + 本行的下三角距离值（含对角线）
  • 矩阵必须为数字、不能包含 NaN，且至少有一个非零距离

4. 参数说明

参数 说明
Input 进化距离文件路径
Distance based method 距离法构树算法：Neighbor-Joining（默认）、Swift Neighbor-Joining 或 UPGMA
Genetic code / IUPAC treatment 与其它距离计算模块共享的全局设置；本模块使用预先计算好的距离矩阵，通常不受该选项影响

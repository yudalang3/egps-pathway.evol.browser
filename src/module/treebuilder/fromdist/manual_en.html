Tree Builder: from Distance Matrix

Reconstruct phylogenetic trees from precomputed evolutionary distance matrices using distance-based methods.

1. Quick Start

  1. View the required input data in the parameter input area
  2. Enter the required parameters
  3. Click Execute to see the results

2. Overview

This module reconstructs a phylogenetic tree from a precomputed evolutionary distance matrix. It supports three distance-based reconstruction methods (Neighbor-Joining, Swift Neighbor-Joining, UPGMA) and roots the final tree automatically using the mid-point method. The result opens in the Modern Tree Viewer.

3. Input Format

The distance file must follow the eGPS lower-triangular matrix format (diagonal included). Blank lines and lines starting with # are ignored.

3.1 Format Example

# number of taxa
3
sp1 0
sp2 0.12 0
sp3 0.20 0.18 0

3.2 Format Rules

  • The first value is the number of taxa (N)
  • Each subsequent line contains a taxon name followed by N values for that row (only the lower triangle is read)
  • Values must be numeric, contain no NaN, and not all be zero

4. Parameters

Parameter Description
Input Path to the evolutionary distance file
Distance based method Neighbor-Joining (default), Swift Neighbor-Joining, or UPGMA
Genetic code / IUPAC treatment Shared global settings for distance calculation in other modules; they do not modify an already computed distance matrix here

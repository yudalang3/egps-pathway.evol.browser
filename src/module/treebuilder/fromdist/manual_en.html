Tree builder: from Dist

Task to resolve

This module reconstructs a phylogenetic tree from a precomputed evolutionary distance matrix. It supports three distance‑based reconstruction methods (Neighbor‑Joining, Swift Neighbor‑Joining, UPGMA) and roots the final tree automatically using the mid‑point method. The result opens in the Modern Tree Viewer.

Input format

The distance file must follow the eGPS lower‑triangular matrix format (diagonal included). Blank lines and lines starting with # are ignored.

# number of taxa
3
sp1 0
sp2 0.12 0
sp3 0.20 0.18 0

The first value is the number of taxa (N). Each subsequent line contains a taxon name followed by N values for that row (only the lower triangle is read). Values must be numeric, contain no NaN, and not all be zero.

Parameters

  1. Input: path to the evolutionary distance file.
  2. Distance based method: Neighbor‑Joining (default), Swift Neighbor‑Joining, or UPGMA.
  3. Genetic code / IUPAC treatment: shared global settings for distance calculation in other modules; they do not modify an already computed distance matrix here.

Quick start

Any actions are straightforward. Please follow the GUI instructions.

  1. View the required input data in the parameter input area.
  2. The user enters the required parameters.
  3. Click the Execute button to see the results.

Gene to Gene Tree

Task to resolve

This pipeline retrieves orthologous sequences from Ensembl/UCSC/eGPS cloud using a gene symbol or genomic region, builds a multiple sequence alignment, computes evolutionary distances, and reconstructs a phylogenetic tree. Tree reconstruction supports Neighbor‑Joining, Swift Neighbor‑Joining, and UPGMA with optional bootstrap. The final tree opens in the Modern Tree Viewer.

Inputs

  1. Web query gene symbol: a gene name such as TP53.
  2. Web query genomic region: an interval such as 17:7661779-7687550 (chr:start-end). Either gene symbol or region is required.

Parameters

  1. Species set for MSA: choose the ortholog set (e.g., Mammals, Primates, Fish, Sauropsids, Amniotes, or eGPS Cloud 1000 species) and a reference genome from the dropdown.
  2. Considered gene regions: align Exon, Intron, CDS, 4‑fold sites, or Whole gene.
  3. Web connection: set REST timeout (3–60 seconds).
  4. Substitution model / Gaps & missing data: choose Model/Method and deletion strategy for distance calculation.
  5. Variance estimation: enable Bootstrap method and set No. of Bootstrap replications for support values.
  6. Distance based method: Neighbor‑Joining (default), Swift Neighbor‑Joining, or UPGMA.
  7. Genetic code / IUPAC treatment: controls ambiguous base handling.

Note: this module requires network access to the selected web resources.

Quick start

Any actions are straightforward. Please follow the GUI instructions.

  1. Enter a gene symbol or genomic region.
  2. Select a species set and reference genome.
  3. Adjust distance/tree/bootstrapping settings if needed.
  4. Click the Execute button to see the results.

Gene to Gene Tree

Retrieve orthologous sequences and reconstruct phylogenetic trees from gene symbols or genomic regions.

1. Quick Start

  1. Enter a gene symbol or genomic region
  2. Select a species set and reference genome
  3. Adjust distance/tree/bootstrapping settings if needed
  4. Click Execute to see the results

2. Overview

This pipeline retrieves orthologous sequences from Ensembl/UCSC/eGPS cloud using a gene symbol or genomic region, builds a multiple sequence alignment, computes evolutionary distances, and reconstructs a phylogenetic tree. Tree reconstruction supports Neighbor-Joining, Swift Neighbor-Joining, and UPGMA with optional bootstrap. The final tree opens in the Modern Tree Viewer.

Note: This module requires network access to the selected web resources.

3. Inputs

Input Description
Web query gene symbol A gene name such as TP53
Web query genomic region An interval such as 17:7661779-7687550 (chr:start-end). Either gene symbol or region is required.

4. Parameters

Parameter Description
Species set for MSA Choose the ortholog set (e.g., Mammals, Primates, Fish, Sauropsids, Amniotes, or eGPS Cloud 1000 species) and a reference genome from the dropdown
Considered gene regions Align Exon, Intron, CDS, 4-fold sites, or Whole gene
Web connection Set REST timeout (3-60 seconds)
Substitution model / Gaps & missing data Choose Model/Method and deletion strategy for distance calculation
Variance estimation Enable Bootstrap method and set No. of Bootstrap replications for support values
Distance based method Neighbor-Joining (default), Swift Neighbor-Joining, or UPGMA
Genetic code / IUPAC treatment Controls ambiguous base handling

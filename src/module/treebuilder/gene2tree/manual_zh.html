从基因到基因树

根据基因符号或基因组区间获取同源序列并重建系统发育树。

1. 快速开始

  1. 输入基因符号或基因组区间
  2. 选择物种集合与参考基因组
  3. 按需调整距离/构树/bootstrap 参数
  4. 点击 执行 查看结果

2. 功能概述

本流程根据基因符号或基因组区间从 Ensembl/UCSC/eGPS cloud 获取同源序列，生成多序列比对（MSA），计算进化距离并重建系统发育树。支持 Neighbor-Joining、Swift Neighbor-Joining、UPGMA 三种距离法构树，并可选 bootstrap 支持度。结果会在 Modern Tree Viewer 中打开。

注意： 本模块需要联网访问所选的在线数据库。

3. 输入

输入 说明
Web query gene symbol 基因名，例如 TP53
Web query genomic region 基因组区间，例如 17:7661779-7687550（chr:start-end）。两者至少填一个。

4. 参数说明

参数 说明
Species set for MSA 选择同源物种集合（如 Mammals、Primates、Fish、Sauropsids、Amniotes 或 eGPS Cloud 1000 species），并在下拉框中选择参考基因组
Considered gene regions 选择比对区域：Exon、Intron、CDS、4-fold 或 Whole gene
Web connection 设置网络超时（3-60 秒）
Substitution model / Gaps & Missing data 选择距离模型（Model/Method）及缺口/缺失处理方式
Variance estimation 若需要 bootstrap 支持度，选择 Bootstrap method 并设置 No. of Bootstrap replications
Distance based method 构树算法：Neighbor-Joining（默认）、Swift Neighbor-Joining 或 UPGMA
Genetic code / IUPAC treatment 控制模糊碱基的处理方式

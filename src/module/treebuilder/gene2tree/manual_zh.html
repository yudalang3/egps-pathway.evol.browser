从基因到基因树

解决的问题

本流程根据基因符号或基因组区间从 Ensembl/UCSC/eGPS cloud 获取同源序列，生成多序列比对（MSA），计算进化距离并重建系统发育树。支持 Neighbor‑Joining、Swift Neighbor‑Joining、UPGMA 三种距离法构树，并可选 bootstrap 支持度。结果会在 Modern Tree Viewer 中打开。

输入

  1. Web query gene symbol：基因名，例如 TP53。
  2. Web query genomic region：基因组区间，例如 17:7661779-7687550（chr:start-end）。两者至少填一个。

参数说明

  1. Species set for MSA：选择同源物种集合（如 Mammals、Primates、Fish、Sauropsids、Amniotes 或 eGPS Cloud 1000 species），并在下拉框中选择参考基因组。
  2. Considered gene regions：选择比对区域：Exon、Intron、CDS、4‑fold 或 Whole gene。
  3. Web connection：设置网络超时（3–60 秒）。
  4. Substitution model / Gaps & Missing data：选择距离模型（Model/Method）及缺口/缺失处理方式。
  5. Variance estimation：若需要 bootstrap 支持度，选择 Bootstrap method 并设置 No. of Bootstrap replications。
  6. Distance based method：构树算法：Neighbor‑Joining（默认）、Swift Neighbor‑Joining 或 UPGMA。
  7. Genetic code / IUPAC treatment：控制模糊碱基的处理方式。

注意：本模块需要联网访问所选的在线数据库。

快速开始

所有操作都比较直观、易懂。请按照图形用户界面的指示进行操作。

  1. 输入基因符号或基因组区间。
  2. 选择物种集合与参考基因组。
  3. 按需调整距离/构树/bootstrapping 参数。
  4. 点击执行按钮查看结果。

进化树构建器：从MSA

解决的问题

本模块用于从多序列比对（MSA）文件重建系统发育树。程序会先根据比对结果计算进化距离（由所选距离模型和缺口/缺失处理方式决定），再用距离法构树（Neighbor‑Joining、Swift Neighbor‑Joining 或 UPGMA）。最终树自动 mid‑point 定根，并在 Modern Tree Viewer 中打开。

输入格式

输入一个已经对齐的 MSA 文件，并在 Data type 下拉框中选择对应格式（下拉框中列出全部支持格式）。比对序列应等长，可包含缺口符号 -。

参数说明

  1. Input：MSA 文件路径。
  2. Data type：选择文件格式（如 aligned FASTA、CLUSTAL、PHYLIP 等，以界面列表为准）。
  3. Substitution model：选择进化距离模型（Model/Method）。
  4. Gaps/Missing Data Treatment：缺口/缺失处理：Complete deletion、Pairwise deletion 或 Partial deletion（可设置 Site Coverage Cutoff）。
  5. Variance Estimation Method：若需要 bootstrap 支持度，选择 Bootstrap method 并设置 No. of Bootstrap replications，支持度会标注在内部节点上。
  6. Distance based method：构树算法：Neighbor‑Joining（默认）、Swift Neighbor‑Joining 或 UPGMA。
  7. Genetic code / IUPAC treatment：控制距离计算时对模糊碱基的处理方式。

快速开始

所有操作都比较直观、易懂。请按照图形用户界面的指示进行操作。

  1. 加载对齐好的 MSA 文件并选择正确的格式。
  2. 按需调整距离与构树参数。
  3. 点击执行按钮查看结果。

进化树构建器：从MSA

从多序列比对（MSA）文件重建系统发育树。

1. 快速开始

  1. 加载对齐好的 MSA 文件并选择正确的格式
  2. 按需调整距离与构树参数
  3. 点击 执行 查看结果

2. 功能概述

本模块用于从多序列比对（MSA）文件重建系统发育树。程序会先根据比对结果计算进化距离（由所选距离模型和缺口/缺失处理方式决定），再用距离法构树（Neighbor-Joining、Swift Neighbor-Joining 或 UPGMA）。最终树自动 mid-point 定根，并在 Modern Tree Viewer 中打开。

3. 输入格式

输入一个已经对齐的 MSA 文件，并在 Data type 下拉框中选择对应格式。比对序列应等长，可包含缺口符号 -。

3.1 支持的格式

  • Aligned FASTA
  • CLUSTAL
  • PHYLIP
  • 以及下拉框中列出的其他格式

4. 参数说明

参数 说明
Input MSA 文件路径
Data type 选择文件格式（如 aligned FASTA、CLUSTAL、PHYLIP 等）
Substitution model 选择进化距离模型（Model/Method）
Gaps/Missing Data Treatment 缺口/缺失处理：Complete deletion、Pairwise deletion 或 Partial deletion（可设置 Site Coverage Cutoff）
Variance Estimation Method 若需要 bootstrap 支持度，选择 Bootstrap method 并设置 No. of Bootstrap replications
Distance based method 构树算法：Neighbor-Joining（默认）、Swift Neighbor-Joining 或 UPGMA
Genetic code / IUPAC treatment 控制距离计算时对模糊碱基的处理方式

Tree builder: from MSA

Task to resolve

This module reconstructs a phylogenetic tree from a multiple sequence alignment (MSA) file. It computes evolutionary distances from the alignment using the selected substitution model and gap/missing‑data treatment, then builds a distance‑based tree (Neighbor‑Joining, Swift Neighbor‑Joining, or UPGMA). The tree is mid‑point rooted and opened in the Modern Tree Viewer.

Input format

Provide an aligned MSA file. Choose the matching format in Data type (the dropdown lists all supported formats). Sequences should be aligned to equal length and may contain gaps (-).

Parameters

  1. Input: path to the MSA file.
  2. Data type: select the file format (e.g., aligned FASTA, CLUSTAL, PHYLIP, etc.).
  3. Substitution model: choose Model/Method for evolutionary distance.
  4. Gaps/Missing Data Treatment: Complete deletion, Pairwise deletion, or Partial deletion (with Site Coverage Cutoff).
  5. Variance Estimation Method: set to Bootstrap method and specify No. of Bootstrap replications to get bootstrap supports on internal nodes.
  6. Distance based method: Neighbor‑Joining (default), Swift Neighbor‑Joining, or UPGMA.
  7. Genetic code / IUPAC treatment: controls how ambiguous bases are handled during distance calculation.

Quick start

Any actions are straightforward. Please follow the GUI instructions.

  1. Load your aligned MSA file and select its format.
  2. Adjust distance and tree parameters if needed.
  3. Click the Execute button to see the results.

Tree Builder: from MSA

Reconstruct phylogenetic trees from multiple sequence alignment (MSA) files.

1. Quick Start

  1. Load your aligned MSA file and select its format
  2. Adjust distance and tree parameters if needed
  3. Click Execute to see the results

2. Overview

This module reconstructs a phylogenetic tree from a multiple sequence alignment (MSA) file. It computes evolutionary distances from the alignment using the selected substitution model and gap/missing-data treatment, then builds a distance-based tree (Neighbor-Joining, Swift Neighbor-Joining, or UPGMA). The tree is mid-point rooted and opened in the Modern Tree Viewer.

3. Input Format

Provide an aligned MSA file. Choose the matching format in Data type (the dropdown lists all supported formats). Sequences should be aligned to equal length and may contain gaps (-).

3.1 Supported Formats

  • Aligned FASTA
  • CLUSTAL
  • PHYLIP
  • And other formats listed in the dropdown

4. Parameters

Parameter Description
Input Path to the MSA file
Data type Select the file format (e.g., aligned FASTA, CLUSTAL, PHYLIP, etc.)
Substitution model Choose Model/Method for evolutionary distance
Gaps/Missing Data Treatment Complete deletion, Pairwise deletion, or Partial deletion (with Site Coverage Cutoff)
Variance Estimation Method Set to Bootstrap method and specify No. of Bootstrap replications to get bootstrap supports on internal nodes
Distance based method Neighbor-Joining (default), Swift Neighbor-Joining, or UPGMA
Genetic code / IUPAC treatment Controls how ambiguous bases are handled during distance calculation

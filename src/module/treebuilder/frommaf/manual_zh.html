进化树构建器：从MAF文件

解决的问题

本模块用于从 UCSC MAF（Multiple Alignment Format）块中重建系统发育树。程序会解析所有 MAF block，基于 block 计算进化距离，然后用距离法构树（Neighbor‑Joining、Swift Neighbor‑Joining 或 UPGMA）。构建后的树会自动进行 mid‑point 定根，并在 Modern Tree Viewer 中打开。

输入格式

Input 需要选择一个包含 MAF 文件的目录。程序会读取目录下所有以 .maf 结尾的文件；若勾选 Is Gzip，则读取 .maf.gz 文件。文件需为标准 MAF 格式，block 之间以空行分隔，序列行以 s 开头：

a score=...
s hg38.chr1  ... ACTG--A...
s panTro4.chr1 ... ACTG--A...
s ponAbe2.chr1 ... ACTG--A...

只有同时包含所有目标基因组的 block 会被用于计算。在当前 GUI 中目标基因组固定为 panTro4、hg38、ponAbe2。

参数说明

  1. Input：包含 MAF 文件的文件夹路径。
  2. Is Gzip：若文件后缀为 .maf.gz 请勾选。
  3. Substitution model / Gaps & Missing data：选择距离模型（Model/Method）以及缺失/缺口处理方式（Complete/Pairwise/Partial deletion）。
  4. Variance estimation：若需要 bootstrap 支持度，选择 Variance Estimation Method = Bootstrap method 并设置 No. of Bootstrap replications。
  5. Distance based method：构树算法：Neighbor‑Joining（默认）、Swift Neighbor‑Joining 或 UPGMA。
  6. Genetic code / IUPAC treatment：控制距离计算时对模糊碱基的处理。

快速开始

所有操作都比较直观、易懂。请按照图形用户界面的指示进行操作。

  1. 选择包含 MAF（或 MAF.GZ）文件的目录。
  2. 根据需要调整距离与构树参数。
  3. 点击执行按钮查看结果。

Tree Tanglegram

Compare two phylogenetic trees by drawing a tanglegram visualization.

1. Quick Start

  1. Provide two tree files in NWK format.
  2. (Optional) Set an outgroup name to re-root both trees before drawing.
  3. Adjust Name font if needed.
  4. Click View tanglegram to display the graph.

2. Notes

  • If no outgroup is provided, the trees are drawn as-is.
  • The outgroup must match a node name in the tree.

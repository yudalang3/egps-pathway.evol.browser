进化树 Tanglegram 可视化

对两棵系统发育树进行比较，绘制 tanglegram 图形。

1. 快速开始

所有操作都比较直观、易懂。请按照图形用户界面的指示进行操作。

详细文档：Yuque 页面

  1. 导入两棵 NWK 格式的树文件。
  2. 可选：设置外类群（outgroup）。若提供外类群名称，程序将先 re-root 再重排后绘制。
  3. 可调整 Name font 获得更理想的显示效果。
  4. 点击 View tanglegram 查看绘图结果。

2. 说明

  • 不提供 outgroup 时，程序按原树结构绘制。
  • outgroup 名称需与树中的节点名称一致。

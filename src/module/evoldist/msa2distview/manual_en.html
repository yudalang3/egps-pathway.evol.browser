MSA to Distance View

Compute and visualize evolutionary distance matrices from MSA files.

1. Quick Start

  1. Click Example to see the input data
  2. Click Execute to see the results

2. Overview

Compute and visualize evolutionary distance matrices from Multiple Sequence Alignment (MSA) files. Supports multiple evolutionary distance models such as JC69, K2P, Tamura-Nei, etc.

3. Main Components

Component Description
MSA2DistanceMatrixViewerMain Main interface for managing MSA input and distance calculation
PLMSAFile2DistMatrix Wizard panel for computing distance matrix from MSA files
PLMSA2DistMatrix Computes and displays evolutionary distance matrices

Task to resolve

Compute and visualize evolutionary distance matrices from Multiple Sequence Alignment (MSA) files. Supports multiple evolutionary distance models such as JC69, K2P, Tamura-Nei, etc.

Quick start

Any actions are straightforward. Please follow the GUI instructions.

  1. Click the Example button to see the input data.
  2. Click the Execute button to see the results.

Main components

  1. MSA2DistanceMatrixViewerMain: Main interface for managing MSA input and distance calculation.
  2. PLMSAFile2DistMatrix: Wizard panel for computing distance matrix from MSA files.
  3. PLMSA2DistMatrix: Computes and displays evolutionary distance matrices.

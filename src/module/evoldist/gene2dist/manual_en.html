Gene to Distance Matrix

Retrieve MSA from web services and compute evolutionary distances.

1. Quick Start

  1. View the required input data in the parameter input area
  2. Enter the required parameters
  3. Click Execute to see the results

2. Overview

Obtain the Multiple Sequence Alignment (MSA) from the Ensembl RESTful service or eGPS cloud and compute the evolutionary distance from the MSA. This module integrates web data retrieval, sequence alignment, and evolutionary analysis functions.

3. Main Components

Component Description
PLWeb2GeneticDist Wizard panel that guides users to retrieve data from web services and compute evolutionary distances
PLWeb2AlignmentView Displays alignment results obtained from the web
DistMatrixViewMainForGene2Dist Visualizes the evolutionary distance matrix

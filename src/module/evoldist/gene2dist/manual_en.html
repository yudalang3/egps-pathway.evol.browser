Task to resolve

Obtain the Multiple Sequence Alignment (MSA) from the Ensembl RESTful service or eGPS cloud and compute the evolutionary distance from the MSA. This module integrates web data retrieval, sequence alignment, and evolutionary analysis functions.

Quick start

Any actions are straightforward. Please follow the GUI instructions.

  1. View the required input data in the parameter input area.
  2. The user enters the required parameters.
  3. Click the Execute button to see the results.

Main components

  1. PLWeb2GeneticDist: A wizard panel that guides users to retrieve data from web services and compute evolutionary distances.
  2. PLWeb2AlignmentView: Displays alignment results obtained from the web.
  3. DistMatrixViewMainForGene2Dist: Visualizes the evolutionary distance matrix.

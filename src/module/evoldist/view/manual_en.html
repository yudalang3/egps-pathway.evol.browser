Distance Matrix View

Visualize symmetric evolutionary distance matrices with heatmap and table displays.

1. Quick Start

  1. Click Example to see the input data
  2. Click Execute to see the results

2. Overview

A visualization module for symmetric evolutionary distance matrices. Provides heatmap and data table displays with interactive viewing and export capabilities.

3. Features

  • Supports symmetric matrix format for evolutionary distance data
  • Provides multiple visualization perspectives: heatmap, numerical table
  • Supports interactive operations such as zooming and clicking for details

Ambiguous Base Expander

Expand sequences with IUPAC ambiguous bases into all possible concrete sequences.

1. Quick Start

  1. View the required input data in the parameter input area (click Example to see more examples)
  2. Enter the required parameters
  3. Click Execute to see the results

2. Overview

A convenient graphic tool to make the nucleotide sequence to concrete sequence. This module provides two core functions:

2.1 Ambiguous Nucleotide Base Enumeration

Expands sequences containing IUPAC ambiguous base codes into all possible concrete sequences.

2.2 DNA Complement Generation

Computes the reverse complement of a given DNA sequence.

4. Technical Notes

This module uses a recursive algorithm to expand ambiguous bases, generating all possible base combinations for each position's ambiguous code. Output is in FASTA format with automatically numbered sequence names.

Task to resolve

A convenient graphic tool to make the nucleotide sequence to concrete sequence. This module provides two core functions:

  1. Ambiguous Nucleotide Base Enumeration (AmbiguousNuclBase): Expands sequences containing IUPAC ambiguous base codes into all possible concrete sequences. Supported ambiguous bases include: R(A/G), Y(C/T), M(A/C), K(G/T), S(C/G), W(A/T), H(A/C/T), B(C/G/T), V(A/C/G), D(A/G/T), N(A/C/G/T).
  2. DNA Complement Generation (DNAComplement): Computes the reverse complement of a given DNA sequence.

Quick start

All user actions are straightforward. Please follow the GUI instructions.

  1. View the required input data in the parameter input area. When the module provides more than one example, you can click the example button to see the next.
  2. The user enters the required parameters.
  3. Click the Execute button to see the results.

Technical notes

This module uses a recursive algorithm to expand ambiguous bases, generating all possible base combinations for each position's ambiguous code. Output is in FASTA format with automatically numbered sequence names.

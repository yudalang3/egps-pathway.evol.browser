MAFFT Aligner

Multiple sequence alignment using the MAFFT program.

1. Quick Start

  1. View the required input data in the parameter input area
  2. Enter the required parameters
  3. Click Execute to see the results

2. Overview

Make multiple sequence alignment by the MAFFT program. MAFFT is a fast and accurate multiple sequence alignment tool suitable for both nucleotide and protein sequences.

3. Important Note

Warning: Please do not install the eGPS2 software in a non-English path, otherwise the alignment tool MAFFT will not function properly.

Task to resolve

Make multiple sequence alignment by the MAFFT program. MAFFT is a fast and accurate multiple sequence alignment tool suitable for both nucleotide and protein sequences.

Quick start

Any actions are straightforward. Please follow the GUI instructions.

  1. View the required input data in the parameter input area.
  2. The user enters the required parameters.
  3. Click the Execute button to see the results.

Note

Please do not install the eGPS2 software in a non-English path, otherwise the alignment tool MAFFT will not function properly.

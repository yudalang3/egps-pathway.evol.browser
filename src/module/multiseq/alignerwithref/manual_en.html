MAFFT Aligner with Reference

Align sequences with a reference sequence placed at the top.

1. Quick Start

  1. View the required input data in the parameter input area
  2. Enter the required parameters
  3. Click Execute to see the results

2. Overview

A convenient module to invoke the MAFFT aligner to align reference with other sequences. This module can place the reference sequence at the top of the alignment.

3. Important Note

Warning: Please do not install the eGPS2 software in a non-English path, otherwise the alignment tool MAFFT will not function properly.

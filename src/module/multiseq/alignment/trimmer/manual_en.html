Problems Addressed

The function of this module is to remove the sequences at both ends of the MSA file based on the reference sequence.

Quick Start

All operations are straightforward and easy to understand. Please follow the instructions provided by the graphical user interface.

  1. After entering the module, input the MSA file in fasta format into the Data import section.
  2. Click the Initialize button to initialize the relevant information.
  3. Then, enter the corresponding reference sequence name and the start and end positions of the sequence.
  4. Finally, click the Run button to execute this module.

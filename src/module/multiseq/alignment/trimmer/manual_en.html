Alignment Trimmer

Trim MSA sequences based on reference sequence positions.

1. Quick Start

  1. Enter the module and input the MSA file in FASTA format in the Data import section
  2. Click Initialize to initialize the relevant information
  3. Enter the corresponding reference sequence name and the start and end positions
  4. Click Run to execute

2. Overview

The function of this module is to remove the sequences at both ends of the MSA file based on the reference sequence. It can trim the multiple sequence alignment results according to the specified reference sequence and position range.

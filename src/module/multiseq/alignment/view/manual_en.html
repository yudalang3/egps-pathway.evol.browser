Alignment View

Interactive visualization of multiple sequence alignments with color schemes and consensus analysis.

1. Quick Start

  1. Click Example to see the input data
  2. Click Execute to see the results

2. Overview

A convenient module to describe the diversity of the alignment. Provides interactive visualization of multiple sequence alignments with various color schemes and consensus analysis.

3. Usage Instructions

  • Data is displayed in blocks; the number of sequences shown depends on font size and window size. Adjust font size using toolbar buttons.
  • Hovering over a base shows the corresponding column number. Clicking a base turns its background red.
  • Base coloring schemes are provided on the left panel.
  • A consensus bar chart is shown at the bottom; hover to see percentages.

4. Supported File Formats

File names are case-insensitive:

  • ClustalW
  • Aligned FASTA
  • GCG MSF
  • PAML
  • MEGA
  • PHYLIP
  • NEXUS

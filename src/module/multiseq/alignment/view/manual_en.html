A convenient module to describe the diversity of the alignment. Provides interactive visualization of multiple sequence alignments with various color schemes and consensus analysis.

Quick start

Any actions are straightforward. Please follow the GUI instructions.

  1. Click the Example button to see the input data.
  2. Click the Execute button to see the results.

Some operational instructions:

  1. Data will be displayed in blocks, and the number of sequences displayed in each block depends on the current font size and window size. You can adjust the font size using the buttons on the toolbar.
  2. When the mouse hovers over a base, the screen will display the corresponding column number for the current base. Clicking on each base will turn the base's background red.
  3. A base coloring scheme is provided on the left, and the coloring effect is as described in the string.
  4. Below is a bar chart displaying the consensus of each column. The percentage can be displayed when the mouse hovers over it.

Supported file formats

The name is case-insensitive.

  1. ClustalW
  2. Aligned fasta
  3. GCG MSF
  4. PAML
  5. MEGA
  6. PHYLIP
  7. NEXUS

Note:

This module has not yet implemented command-line operation and requires some modification.

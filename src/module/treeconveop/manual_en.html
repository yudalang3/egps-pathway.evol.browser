Phylogenetic Tree Operator

A suite of tools for rapid operations on phylogenetic trees.

1. Quick Start

  1. Click Import to open the data import dialog.
  2. Check the required inputs in the parameter panel (use Example to switch examples when available).
  3. Fill in the required parameters.
  4. Click Execute to get the results.

2. Functions

  • Subtree extraction - Extract a subtree based on a target node.
  • Node information - Obtain basic information for nodes/leaves.

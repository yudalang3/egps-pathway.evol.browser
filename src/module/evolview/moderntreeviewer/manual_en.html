Task to resolve

Interactive visualization and operation of phylogenetic trees. Supports multiple tree layouts, color configurations, and graphical annotations with high-quality vector graphics export.

Quick start

Any actions are straightforward. Please follow the GUI instructions.

  1. Click the Example button to see the input data.
  2. Click the Execute button to see the results.

The novel features in the control panel (GUI):

  1. A modern graphical interface, not the previous dull state. It can remember some user operations.
  2. Control panel buttons, divided into four categories:
    1. First category: Gesture buttons
    2. Second category: Normal buttons
    3. Third category: Counting gesture buttons
    4. Fourth category: Swing Toggle buttons
  3. Highly recommended is this Toggle button with an Apple desktop style, which is used in the Tree layout and Creative mode panels.
  4. VOICM solves the parameter explosion issue:
    1. Bookmarks that can be recorded, support free operations, and can be imported and exported.
    2. All parameters can be set in the input panel.

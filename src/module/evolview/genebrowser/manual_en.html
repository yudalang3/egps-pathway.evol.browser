Gene Browser

Visualize single-gene structure and related tracks (mRNA, protein, exons/introns, etc.).

1. Quick Start

  1. Click Example to view the input data format.
  2. Click Import to load your own data.
  3. Click Execute to render the tracks.

2. Notes

  • The supported fields and parameter meanings are described in the VOICE input dialog.
  • Use the control panel to zoom/scroll and manage tracks.

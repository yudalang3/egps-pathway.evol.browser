Tree with Barplot

Plot a combined figure: phylogenetic tree on the left and bar plot on the right (with a geological time scale legend).

1. Quick Start

  1. Click Import to load the data.
  2. Set the required parameters in the VOICE input dialog (parameter descriptions are provided there).
  3. Click Execute to render the plot.

2. Tips

  • For control panel usage, refer to the help of Modern tree view module.

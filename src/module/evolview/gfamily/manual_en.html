Gene Family Browser

Interactive phylogenetic tree and sequence structure visualization for gene family analysis.

1. Quick Start

  1. Click Example to see the input data
  2. Click Execute to see the results

2. Overview

Highly interactive phylogenetic tree and sequence structure visualization. Displays phylogenetic trees and sequence features simultaneously, facilitating comparative analysis of gene families.

3. Features

  • Displays phylogenetic trees and sequence structural domains simultaneously
  • Supports interactive operations such as node selection and alignment viewing
  • Suitable for gene family evolution analysis and functional domain comparison
